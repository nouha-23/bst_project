# Arbre de Recherche Binaire (BST) en C

## 📖 C'est quoi ?
Ce projet est une implémentation complète d'un Arbre de Recherche Binaire (Binary Search Tree - BST) développée dans le cadre de ma formation en intelligence artificielle. Il propose plusieurs algorithmes de parcours et de manipulation de données structurées, illustrant mes compétences en développement logiciel bas niveau et en conception d'algorithmes et de structures de données.

## 🛠 Les technologies utilisées
- **C** : Langage principal de développement (programmation procédurale / gestion de mémoire).
- **Makefile** : Pour l'automatisation de la compilation.
- **Git** : Pour le versionnage du code.

## 🚀 Comment ça marche ?
Pour compiler et lancer le projet, assurez-vous d'avoir `gcc` (ou `clang`) et `make` installés sur votre machine, puis exécutez les commandes suivantes :

```bash
# Compiler le projet
make

# Lancer l'exécutable
./bin/bst
```
